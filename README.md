# CxRedix's Pixelart Expansion

Just a big expansion to Sharpy796's [Glimmers Expanded](https://github.com/Sharpy796/GlimmersExpanded) mod!


### Prerequisites

1. Download Sharpy796's [Glimmers Expanded](https://github.com/Sharpy796/GlimmersExpanded) :)


### Installation instructions
1. download the zip by clicking on the big blue "<> Code" button, and clicking on "Download ZIP" (if you are downloading from github)

2. extract this into your noita mods' directory.
check this [tutorial from the Noita wiki](https://noita.wiki.gg/wiki/How_to_install_mods#Manual) Go to the "Manual" section :)

3. go to your in game mods list (the place where you view all mods in game), make sure you move this mod UNDER **Glimmers Expanded**.
This is required since Glimmers Expanded has to load first for the mod to extend all the "new" glimmers :)
